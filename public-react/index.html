CSV Image Generator API

How it works:
  1. Upload a CSV file with your data
  2. The API will generate images for each row using the template
  3. Images are uploaded to Wasabi cloud storage
  4. An updated CSV with image links is created and uploaded
  5. You'll receive a download link for the updated CSV
